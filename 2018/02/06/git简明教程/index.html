git 全局忽略的使用

例如这里我们要忽略mac系统自动生成的隐藏文件.DS_Store，那么我们可以执行如下几步操作

  • 创建~/.gitignore_global(如果有则直接打开)，

  • 然后编辑.gitignore_global文件，添加要忽略的文件

  • 把.gitignore_global文件添加到~/.gitconfig中

  • 查看是否配置成功
